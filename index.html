      Settings

      Groq API Key (Free)

      Completely free, no credit card needed. Sign up at console.groq.com → API Keys → Create API Key. Your key starts with gsk_ and is stored on this device only.

      ✓ Groq API key saved on this device
      Categories

      🏷️ Manage Categories

      Add your own categories. Default ones can't be removed, but your custom ones can.

      Cloud Sync

      ☁️ Sync Between Devices — Firebase (Free)

      Uses Google Firebase — free forever, up to 1GB storage. Takes ~2 minutes to set up:

      1. Go to console.firebase.google.com
      2. Create a project → Add a Realtime Database
      3. Start in test mode (or locked mode + copy secret)
      4. Copy your Database URL (looks like https://your-app.firebaseio.com)
      5. Go to Project Settings → Service Accounts → Database secrets → copy your secret
      Backup & Restore

      📤 Export Recipes

      Download all your recipes as a JSON file. Save it somewhere safe as a backup.

      📥 Import Recipes

      Restore from a previously exported backup. New recipes will be added — existing ones won't be duplicated.

      Install as App

      📲 Get the Full App Experience

      To install Cronjes as a real standalone app (with its own icon, no browser bar, and isolated storage that clears properly when uninstalled), the file needs to be hosted online.

      Free in 30 seconds: go to app.netlify.com/drop, drag and drop your index.html file onto the page, then open the given link on your phone.

      Clear Data

      🗑️ Clear All App Data

      Deletes all recipes, your API key, settings, and resets the app. Export a backup first if you want to keep your recipes.
